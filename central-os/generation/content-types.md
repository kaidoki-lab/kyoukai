# KYOUKAI generated content types

## Types

- `image`: room visuals, object visuals, background candidates
- `short-video`: short-form video concepts or cut plans
- `music`: BGM or soundscape concepts
- `text`: short copy, room text, UI-world text
- `room-event`: event idea triggered by room visit or return
- `sns-post`: SNS post draft or concept
- `hyougi-log`: council record / 評議録 candidate

## Status

- `候補`: idea exists, not reviewed
- `確認待ち`: needs human check
- `採用`: approved for use
- `使用済み`: already used in KYOUKAI
- `却下`: rejected
- `保留`: parked for later
