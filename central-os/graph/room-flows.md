# KYOUKAI 回遊構造定義

KYOUKAI内部辞書 — 部屋間の流れ・回遊構造の定義

---

## 基本思想

KYOUKAIは「部屋単体」で完結しない。
「どこからどこへ流れるか」という導線が、世界観の深度を作る。

訪問者は意図せず深層へ引き込まれ、気づけば観測されている。

---

## 初見導線

初回訪問者が辿る標準的な経路。

```
/ (祭壇域)
  ↓
/observation (観測域)
  ↓
/hyougi (評議録)
```

### 意図

- 祭壇域でKYOUKAIの存在を感知する
- 観測域で「何かを観測している感覚」を得る
- 評議録でKYOUKAI内部の記録に触れる
- この段階では異常感は低い

---

## 深層導線

繰り返し訪問・長時間滞在で開かれる経路。

```
/observation (観測域)
  ↓
/signal (受信域)
  ↓
/null (未確認接続)
  ↓
/observer (逆観測室)
```

### 意図

- 観測を続けるうちに「信号」を受信し始める
- 受信域で未知の発信源に気づく
- 未確認接続で「何かに繋がっている」感覚が生まれる
- 逆観測室で「自分が観測されていた」という逆転が起きる

---

## 滞在導線

KYOUKAI内部に長く留まるユーザー向けの周回経路。

```
/hyougi (評議録)
  ↓
/observation (観測域)
  ↓
/hyougi (評議録)  ← 周回
```

### 意図

- 評議録と観測域を往復することで記録量が増える感覚
- 「見るたびに変化している」体験を作る
- 滞在時間を自然に延ばす

---

## 異常感増加ルート

KYOUKAIの「おかしさ」が段階的に強まる経路。

```
/ (祭壇域)
  ↓ [通常]
/observation (観測域)
  ↓ [深層]
/signal (受信域)
  ↓ [不安定]
/null (未確認接続)
  ↓ [隠し]
/observer (逆観測室)
```

### 意図

- 各段階で「異常感」のレベルが上がる
- 段階を経ないと次のフェーズへ進みにくい設計
- 逆観測室は最も深い場所として機能する

---

## 初回訪問者導線

SNS経由で初めて訪問するユーザー向けの設計。

```
SNS (YouTube / TikTok / X)
  ↓
/ (祭壇域)  ← ランディング
  ↓
/observation (観測域)  ← 最初の探索
```

### 意図

- 祭壇域でKYOUKAIが「場所」であることを認識させる
- 離脱せずに観測域まで誘導する
- 収益導線への接触は最初は求めない

---

## 再訪問導線

2回目以降の訪問者が向かう経路。

```
/ (祭壇域)
  ↓ [前回より深い選択]
/signal (受信域)  or  /null (未確認接続)
  ↓
/observer (逆観測室)
```

### 意図

- 再訪問者は祭壇域を通過して深層へ向かう
- 初回に見えなかった導線が見え始める
- 収益導線（外部接続）への接触機会も増える

---

## 収益接触タイミング

```
/observation (観測域) → 滞在時間増加
  ↓
/outside (外部接続)  ← 世界観の「外」として提示
```

- 収益導線は「広告」として提示しない
- 「外部への扉」「世界の外縁」として世界観に溶け込ませる
- 押し付けず、導線を発見させる
