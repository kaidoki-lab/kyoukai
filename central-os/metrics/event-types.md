# KYOUKAI observation event types

## Types

- `page-view`: page display count or planned page observation
- `route-click`: movement from one route to another
- `sns-inflow`: SNS-origin visit pattern
- `deep-arrival`: arrival at a deeper room/layer
- `observer-arrival`: arrival at the observer/reverse-observation concept
- `monetization-touch`: contact with outside/support/monetization boundary
- `generated-used`: generated candidate adoption/use status
- `revisit`: returning visitor or top return behavior

## Note

These types define records only. They do not install tracking.
