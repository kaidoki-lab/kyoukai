# 境界ワールド

## フロア自動増設ルール

KYOUKAIでは1フロアの部屋数を5部屋までに固定する。
同一フロアに6部屋目は存在しない。
6部屋目が追加された場合は、自動的に次フロアの1部屋目として登録する。
例：6階に5部屋ある状態で新しい部屋を追加した場合、その部屋は7階1部屋目になる。
この処理は手動ではなく、部屋追加時に自動で行う。

## 最上階固定部屋ルール

KYOUKAIでは通常フロアとは別に、常に一番上へ配置される最上階専用部屋を持つ。
この部屋は1フロア5部屋固定ルールの対象外とする。
通常部屋がどれだけ追加され、階層が増えても、最上階専用部屋は常に通常部屋の最上階より一つ上の階に配置する。
最上階には同一フロア内の他の部屋や空きスロットを表示しない。
最上階は常にこの専用部屋1つだけで構成する。
