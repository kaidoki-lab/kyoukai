# Dot Collision Art v1.0.0

上下左右の画面端から丸いドットを発射し、壁面またはドット同士の衝突で小さな爆発と痕跡が生まれる、全画面型のインタラクティブ作品です。

## 起動方法

`index.html` をブラウザで開いてください。

ローカルファイルとして開いて動作します。ブラウザ設定や環境により音声が再生されない場合は、作品フォルダで簡易ローカルサーバーを起動してからアクセスしてください。

```bash
python -m http.server 8000
```

起動後、ブラウザで `http://127.0.0.1:8000/` を開きます。

## 動作環境

- 最新版の Chrome、Edge、Firefox、Safari
- スマートフォンおよびPCブラウザ
- JavaScript、Canvas 2D、Pointer Events、Web Audio API が有効な環境

## 操作方法

- 画面上部を触ると、上から下へドットが進みます。
- 画面下部を触ると、下から上へドットが進みます。
- 画面左側を触ると、左から右へドットが進みます。
- 画面右側を触ると、右から左へドットが進みます。
- 押したまま移動すると、一定間隔で連続発射します。
- 右下の「音」ボタンで効果音をON/OFFできます。

## 既知の問題

- iOS Safari など一部ブラウザでは、最初のタップ後に音声が有効になります。
- 端末性能が低い場合、大量入力時に爆発片の数が自動的に減ります。
