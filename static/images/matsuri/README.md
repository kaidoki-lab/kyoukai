# 棒入れ祭 画像素材

「KYOUKAI 棒入れ祭 素材入手担当表 v1.0」のフォルダ構成に対応。
各サブフォルダに、対応するファイル名でPNGを置けば `static/matsuri.js` が自動的に読み込む。
ファイルが無い場合はCSSのプレースホルダー（色面）で表示されるので、実装の動作確認に画像は必須ではない。
