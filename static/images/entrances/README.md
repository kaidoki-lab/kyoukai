# KYOUKAI home entrance images

Image folder for the temporary home entrance strip.

After adding an image, set the matching `image` value in `static/home-entrances.js` to the same path as `expectedImage`. While `image` is empty, the page renders a placeholder object and does not emit a broken image URL.
