【同梱ファイル】

  01_waiting / waiting.html
    待機画面。配信開始前・ゲームロード中などに使用。
    オシロスコープ波形とノイズが流れ続ける。

  02_brb / brb.html
    離席中画面。「接続を試みています」のメッセージが表示される。
    TVスタティックノイズとグリッチ演出入り。

  03_lower-third / lower_third.html
    ローワーサード（名前テロップ）。透過背景。
    他の素材の上に重ねてオーバーレイとして使う。

【OBS設定方法】

  1. OBS Studio を開く
  2. ソース → ＋ → ブラウザ を追加
  3. 「ローカルファイル」にチェックを入れる
  4. 各 .html ファイルを指定する
  5. 幅: 1920  /  高さ: 1080 に設定する
  6. カスタムCSSは空白でOK

【ローワーサードのカスタマイズ】

  lower_third.html をテキストエディタで開き、
  以下の2行を書き換えてください。

    const name  = params.get('name')  || '名前';
    const title = params.get('title') || 'KYOUKAI';

  '名前' → あなたの名前
  'KYOUKAI' → 肩書きや活動名

  例：
    const name  = params.get('name')  || 'まろ';
    const title = params.get('title') || '配信者';
