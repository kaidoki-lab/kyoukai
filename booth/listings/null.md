# 崩落域 OBS素材パック ── 崩壊は止まらない

## 商品説明文

崩落域は、画面自体が傾き、UI部品がずり落ちていく部屋。データはある。だが何のデータかはわからない。ページとして機能しているが、周期的に崩壊と再構築を繰り返す。長く留まると、自分が何をしていたのかも曖昧になる。

【同梱内容】3点

  01_waiting / waiting.html
    待機画面。崩壊 進行中の状態を表示し続ける。
    配信前、ゲームロード中、場面転換に使用する。

  02_brb / brb.html
    離席中画面。「崩落は続いています」というメッセージが表示される。
    離席時、準備中、一時中断時に使用する。

  03_lower-third / lower_third.html
    ローワーサード（名前テロップ）。透過背景。
    他の素材の上に重ねてオーバーレイとして使用する。

【OBS設定方法】

  1. OBS Studio を開く
  2. ソース → ＋ → ブラウザ を追加
  3. 「ローカルファイル」にチェックを入れる
  4. 各 .html ファイルを指定する
  5. 幅: 1920  /  高さ: 1080 に設定する

  ローワーサードの名前・肩書きは lower_third.html 内の該当箇所を
  書き換えることで変更できる。

【動作環境】

  OBS Studio 29以降推奨
  インターネット接続不要 / 完全ローカル動作

【利用規約】

  改変は個人使用の範囲で自由
