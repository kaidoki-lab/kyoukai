# 観測域 OBS素材パック ── 観測は続いている

## 商品説明文

観測域は、KYOUKAIの中でもっとも活動が多い部屋。画面全体を流れる観測ログ端末に、生命体の記録が絶えず積み上がっていく。ログは感情を持たない。だが、それを見ていると何かを感じる。観測対象が誰なのかは、最後まで語られない。

【同梱内容】3点

  01_waiting / waiting.html
    待機画面。観測継続中の状態を表示し続ける。
    配信前、ゲームロード中、場面転換に使用する。

  02_brb / brb.html
    離席中画面。「観測を再開します」というメッセージが表示される。
    離席時、準備中、一時中断時に使用する。

  03_lower-third / lower_third.html
    ローワーサード（名前テロップ）。透過背景。
    他の素材の上に重ねてオーバーレイとして使用する。

【OBS設定方法】

  1. OBS Studio を開く
  2. ソース → ＋ → ブラウザ を追加
  3. 「ローカルファイル」にチェックを入れる
  4. 各 .html ファイルを指定する
  5. 幅: 1920  /  高さ: 1080 に設定する

  ローワーサードの名前・肩書きは lower_third.html 内の該当箇所を
  書き換えることで変更できる。

【動作環境】

  OBS Studio 29以降推奨
  インターネット接続不要 / 完全ローカル動作
