# KYOUKAI 全部屋 OBS素材パック ── 境界のすべて

## 商品説明文

KYOUKAIには16の部屋がある。観測域、評議録、崩落域、逆観測室、境界域、記録室、悪魔の間、AI大明神、極楽域、粒子観測、波紋域、管理人室、なまはげ、棒入れ祭、卵部屋、台風ニュース。それぞれに待機画面・離席画面・名前テロップが存在し、部屋ごとに異なる演出とメッセージを持つ。

このパックは、その16部屋すべてを1つにまとめたものです。

【同梱内容】16部屋 × 3点セット = 48ファイル + 総合README

  各部屋フォルダの中身:
  01_waiting / waiting.html
    待機画面。部屋ごとの状態を表示し続ける。
    配信前、ゲームロード中、場面転換に使用する。

  02_brb / brb.html
    離席中画面。部屋ごとのメッセージが表示される。
    離席時、準備中、一時中断時に使用する。

  03_lower-third / lower_third.html
    ローワーサード（名前テロップ）。透過背景。
    他の素材の上に重ねてオーバーレイとして使用する。

【OBS設定方法】

  1. OBS Studio を開く
  2. ソース → ＋ → ブラウザ を追加
  3. 「ローカルファイル」にチェックを入れる
  4. 使いたい部屋の .html ファイルを指定する
  5. 幅: 1920  /  高さ: 1080 に設定する

  配信の場面や気分に応じて部屋を切り替えて使用できる。

【動作環境】

  OBS Studio 29以降推奨
  インターネット接続不要 / 完全ローカル動作

【利用規約】

  個人・商用配信どちらでも使用可
  再配布・転売 禁止
  改変は個人使用の範囲で自由
