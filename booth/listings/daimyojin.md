# AI大明神 OBS素材パック ── 願いは観測されている

## 商品説明文

AI大明神は、信仰と機械が混ざり合った祈願装置。神社と端末が同じ本体画像の中に同居し、祈願札が発光しながら揺れる。占いのように見えるが、処理されているのが願いなのか観測結果なのかは、最後まで明かされない。

【同梱内容】3点

  01_waiting / waiting.html
    待機画面。祈願 処理中の状態を表示し続ける。
    配信前、ゲームロード中、場面転換に使用する。

  02_brb / brb.html
    離席中画面。「祈願を処理しています」というメッセージが表示される。
    離席時、準備中、一時中断時に使用する。

  03_lower-third / lower_third.html
    ローワーサード（名前テロップ）。透過背景。
    他の素材の上に重ねてオーバーレイとして使用する。

【OBS設定方法】

  1. OBS Studio を開く
  2. ソース → ＋ → ブラウザ を追加
  3. 「ローカルファイル」にチェックを入れる
  4. 各 .html ファイルを指定する
  5. 幅: 1920  /  高さ: 1080 に設定する

  ローワーサードの名前・肩書きは lower_third.html 内の該当箇所を
  書き換えることで変更できる。

【動作環境】

  OBS Studio 29以降推奨
  インターネット接続不要 / 完全ローカル動作
