DOT RIPPLE DESK

画面いっぱいのドットに触れると波紋が広がる、目的のないデジタルガジェットです。

クリック、タップ、ドラッグ、スワイプに反応して、ドットが色を変えながらゆっくり動きます。
作業中の第二モニター、配信待機画面、休憩中のぼんやり時間にどうぞ。

ゲームではありません。
スコアも、クリア条件も、ランキングもありません。
ただ触って、眺めるための小さな画面です。

■ 主な機能

- 全画面ドットグリッド
- クリック / タップで波紋
- ドラッグ / スワイプで連続波紋
- 自動波紋モード
- 5種類の色テーマ
  - Classic
  - Neon
  - Pastel
  - Monochrome
  - Sunset
- ドットサイズ、間隔、波紋速度、明るさなどの設定
- localStorageによる設定保存
- フルスクリーン対応
- たまに少しだけ変な反応をするドット

■ 使い方

ZIPを解凍して、index.html をブラウザで開くだけです。
インストール不要、インターネット接続不要です。

■ キー操作

F: フルスクリーン
S: 設定パネル
R: リセット
Space: 自動波紋ON/OFF

■ 利用可能範囲

購入者本人の個人利用、配信画面、動画背景、作業用画面として使用できます。
収益化済み配信や動画内での使用も可能です。
スクリーンショットや動画投稿も可能です。

■ 禁止事項

- ファイルの再配布
- 改変版の販売
- 素材としての無断転載
- 本ソフトをテンプレートやアプリ部品として再販売すること

■ 推奨環境

Chrome / Edge / Firefox / Safari の最新版
iOS Safari / Android Chrome の最新版

重い場合は、設定からドット間隔を広げる、FPS制限を30にする、残像量を下げると軽くなります。
