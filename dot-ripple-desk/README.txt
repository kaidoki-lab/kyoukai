============================================================
DOT RIPPLE DESK v1.0
============================================================

DOT RIPPLE DESKは、画面いっぱいのドットに触れると波紋が広がる、
目的のないデスクトップガジェットです。

作業中の第二モニター、配信待機画面、休憩中のぼんやり時間に使えます。
ゲームではありません。スコア、クリア条件、ランキングはありません。
ただ触って、眺めるための小さな画面です。

------------------------------------------------------------
起動方法
------------------------------------------------------------

1. ZIPファイルを解凍します。
2. dot-ripple-desk フォルダ内の index.html をブラウザで開きます。
3. そのまま画面をクリック、タップ、ドラッグして使えます。

インストールは不要です。
インターネット接続も不要です。

------------------------------------------------------------
基本操作
------------------------------------------------------------

クリック / タップ:
  その位置から波紋を1つ発生させます。

ドラッグ / スワイプ:
  移動に合わせて波紋を連続発生させます。

F:
  フルスクリーン切り替え

S:
  設定パネル表示 / 非表示

R:
  画面リセット

Space:
  自動波紋 ON / OFF

Esc:
  設定パネルを閉じる

------------------------------------------------------------
設定項目
------------------------------------------------------------

右上の小さな丸ボタン、または S キーで設定パネルを開けます。
設定はブラウザの localStorage に保存されます。
次回開いたときも、前回の設定が維持されます。

設定できる項目:

- 色テーマ
- ドットサイズ
- ドット間隔
- 波紋速度
- 波紋強度
- 自動波紋 ON / OFF
- 自動波紋頻度
- 残像量
- 明るさ
- 異常ドット ON / OFF
- FPS制限
- リセット

------------------------------------------------------------
色テーマ
------------------------------------------------------------

Classic:
  黒背景に赤、青、黄、白を使った標準テーマです。

Neon:
  蛍光色を中心にした明るいテーマです。

Pastel:
  暗めの背景に淡い色が乗る落ち着いたテーマです。

Monochrome:
  黒、白、グレー中心のシンプルなテーマです。

Sunset:
  赤、橙、紫、暗色を中心にしたテーマです。

------------------------------------------------------------
自動波紋モード
------------------------------------------------------------

Auto rippleをONにすると、触っていない間もランダム位置から波紋が発生します。
第二モニターや配信待機画面として使う場合におすすめです。

------------------------------------------------------------
異常ドットについて
------------------------------------------------------------

画面内に、たまに少しだけ変な反応をするドットがあります。
通常の波紋に完全には従いません。
気になる場合は、設定パネルの Strange dot をOFFにできます。

------------------------------------------------------------
推奨環境
------------------------------------------------------------

PC:
  Chrome / Edge / Firefox / Safari の最新版

スマートフォン / タブレット:
  iOS Safari / Android Chrome の最新版

推奨:
  できるだけ新しいブラウザ
  フルスクリーン表示
  配信利用時はFPS制限 30 または 45

------------------------------------------------------------
配信利用・動画利用
------------------------------------------------------------

購入者本人の配信画面、動画背景、待機画面、作業用画面として使用できます。
スクリーンショットや動画投稿も可能です。

OBSなどで使う場合:
  index.html をブラウザソースまたはローカルHTMLとして読み込んでください。
  重い場合は設定でドット間隔を広げる、FPS制限を30にする、残像量を下げると軽くなります。

------------------------------------------------------------
同梱ファイル
------------------------------------------------------------

index.html
  アプリ本体を開くHTMLです。

style.css
  画面と設定パネルのスタイルです。

main.js
  ドット、波紋、設定保存などの動作本体です。

README.txt
  この説明ファイルです。

LICENSE.txt
  利用規約です。

------------------------------------------------------------
注意事項
------------------------------------------------------------

- 長時間表示に対応していますが、端末性能やブラウザによって負荷は変わります。
- ドット数を増やしすぎると動作が重くなる場合があります。
- localStorageを消去すると設定は初期化されます。
- 再配布、改変版の販売、素材としての無断転載は禁止です。
